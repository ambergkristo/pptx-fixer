# Codex Workflow

## Roles

ChatGPT

- hoiab scope
- teeb milestone review
- kontrollib drift

Codex

- kirjutab koodi
- refaktor
- testid

Sina

- jooksutad koodi
- otsustad järgmise sammu

## Rules

1 milestone korraga

Iga milestone:

1. Codex implementation
2. ChatGPT review
3. Fix
4. Signoff

## Drift Prevention

Küsi alati:

Kas see töö toetab aktiivset milestone'i?

Kui ei, ära tee seda.
