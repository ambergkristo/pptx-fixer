# PPTX FIXER Interview Script

## Purpose

Use this script to validate whether formatting cleanup in existing PowerPoint decks is a frequent and valuable problem to solve safely.

## Interview Questions

1. What is your role, and how often do you work with PowerPoint files?
2. What kinds of PowerPoint decks do you usually create, edit, or review?
3. How often do you need to clean up formatting in an existing `.pptx` deck?
4. What formatting issues do you run into most often?
5. Which issues are the most frustrating or time-consuming to fix?
6. Roughly how much time do you spend on cleanup for a typical deck?
7. What is your current workflow when you notice formatting inconsistencies?
8. Are there any repetitive cleanup tasks you wish you could avoid?
9. If there were a safe tool that fixed high-confidence formatting issues without changing slide meaning or editability, would you try it? Why or why not?
10. What trust, privacy, or file-safety concerns would you have before using a tool like this?

## Interviewer Notes

- Ask for a recent example of a messy deck when possible.
- Prefer concrete examples over abstract opinions.
- Capture exact language around trust, privacy, and safety concerns.
